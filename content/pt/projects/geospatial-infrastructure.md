# Infraestrutura de dados espaciais e gestão territorial

> [English version](../../en/projects/geospatial-infrastructure.md)

## Visão geral

Projeto voltado à estruturação e automação de soluções geoespaciais aplicadas à gestão territorial, com ênfase em bancos de dados espaciais, metadados, interoperabilidade e serviços geoespaciais.

## Por que importa

Ambientes institucionais e projetos territoriais dependem de bases consistentes, rastreáveis e interoperáveis. Esse eixo conecta a dimensão técnica da infraestrutura geoespacial com necessidades reais de integração e governança de dados.

## Escopo

- modelagem de bancos de dados geográficos;
- integração de dados multi-origem;
- padronização cartográfica e semântica;
- controle de qualidade e validação técnica;
- gestão e serialização de metadados;
- interoperabilidade de serviços geoespaciais.

## Ferramentas e padrões

- PostgreSQL / PostGIS
- GeoServer / GeoNode
- QGIS / Python / Jupyter Notebook / PowerShell
- ET-EDGV / Perfil MGB / ISO 19115 / OGC
