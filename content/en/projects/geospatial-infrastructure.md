# Spatial data infrastructures and territorial data management

> [Versão em português](../../pt/projects/geospatial-infrastructure.md)

## Overview

Project focused on structuring and automating geospatial solutions for territorial data management, with emphasis on spatial databases, metadata, interoperability and geospatial web services.

## Why it matters

Institutional environments and territorial projects depend on consistent, traceable and interoperable datasets. This strand connects the technical dimension of geospatial infrastructure with real needs for data integration and governance.

## Scope

- geographic database modelling;
- multi-source data integration;
- cartographic and semantic standardization;
- technical quality control and validation;
- metadata management and serialization;
- interoperability of geospatial services.

## Tools and standards

- PostgreSQL / PostGIS
- GeoServer / GeoNode
- QGIS / Python / Jupyter Notebook / PowerShell
- ET-EDGV / Brazilian MGB Profile / ISO 19115 / OGC
